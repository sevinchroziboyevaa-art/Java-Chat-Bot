# Java ChatBot Assistant 🤖

An interactive console-based ChatBot that demonstrates core Java programming concepts. This project was developed as part of the **JetBrains Academy** Java curriculum.

## ✨ Features
- **Personalized Greeting:** Asks for the user's name and interacts politely.
- **Age Guesser:** Uses a mathematical algorithm (remainders of division by 3, 5, and 7) to guess the user's age.
- **Counting Tool:** Proves its counting ability by iterating up to any number the user provides.
- **Knowledge Test:** Includes an interactive multiple-choice quiz about programming methods.

## 🛠 Tech Stack
- **Language:** Java 11+
- **Key Concepts:** Static Methods, For-Loops, While-Loops, Scanner (User Input), Conditional Logic (If-Else).

## 🚀 How to Run
1. Ensure you have Java installed.
2. Clone the repository.
3. Run the following commands:
   ```bash
   javac src/bot/SimpleBot.java
   java bot.SimpleBot
   ```
